# rocket_api
Start creating gems with rocket_api
- Add structure of gem repo by one commands
- Init gems repo by files
## Dependencies:
```sh
ruby '2.7.2'
...
gem 'rubocop' '1.26'
```
## Install:
```sh
gem install rocket_api
```
## How it works:
```sh   
 Availble commands:
 - init
 - gem [ project_name ]
 In working directory:
 
 user@bulkas ~/gem/new_dir rocket_api [ init ] [ gem ] [ project_name ]
```
